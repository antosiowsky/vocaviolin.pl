# vocaviolin.pl

Strona internetowa dla Michaliny Porszkę – wokalistki i skrzypaczki, oferującej oprawę muzyczną ślubów, ceremonii i eventów na Śląsku.

## Funkcje
- Responsywny design (Tailwind CSS)
- Sekcje: O mnie, Oferta, Nagrania, Opinie, Kontakt
- Formularz kontaktowy (PHP, walidacja, obsługa błędów)
- Własne fonty i logo
- Favicon (SVG/PNG)
- Dostępność: aria-label, lazy loading, focus styles

## Uruchomienie lokalne
1. Wymagany serwer PHP (np. XAMPP, WAMP, `php -S localhost:8000`)
2. Otwórz stronę w przeglądarce: `http://localhost:8000`

## Struktura projektu
- `index.html` – główna strona
- `send.php` – obsługa formularza kontaktowego
- `assets/css/style.css` – style
- `assets/fonts/` – czcionki
- `assets/images/` – obrazy, favicon
- `assets/js/` – (opcjonalnie) skrypty JS
